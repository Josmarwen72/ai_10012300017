# Hugging Face Spaces Deployment

## Quick Setup for Flask RAG App

### 1. Create a new Space
- Go to https://huggingface.co/spaces
- Click "Create new Space"
- Choose "Flask" as SDK
- Make it Public (free tier)

### 2. Requirements
- Create `requirements.txt` (already exists)
- Upload all files to the Space
- Hugging Face will auto-deploy

### 3. Benefits
- **Free forever** for public spaces
- **ML-friendly** environment
- **GPU options** available
- **No payment** required for public repos

### 4. Limitations
- Public code (visible to everyone)
- Some resource limits
- May need to optimize for memory

### 5. Deployment URL
Your app will be available at: `https://yourusername-yourrepo.hf.space`
